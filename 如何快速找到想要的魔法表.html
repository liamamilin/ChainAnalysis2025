Chapter 13 如何快速找到想要的魔法表

在 Dune Analytics 中，想要快速找到你需要的 魔法表（Spells），可以使用以下方法：

13.1 📌 1. 直接在 Dune 的 “Data Explorer” 搜索

最快捷的方法是： 1. 打开 Dune 2. 进入 “Data Explorer” 3. 在搜索框输入关键词 - 如果你知道表的名称（例如 uniswap_v3_ethereum.swaps），直接搜索即可。 - 如果你不知道确切名称，可以使用代币名称、合约地址、协议名称搜索（如 “AAVE”, “ERC20”, “USDT”）。 - 使用 dbt_subprojects、“spellbook” 关键词，可以查找 Spellbook 相关的表。

📌 示例搜索： - 查找 Uniswap 交易数据 → 搜索 uniswap - 查找 ERC-20 代币数据 → 搜索 erc20 - 查找 Aave 相关数据 → 搜索 aave - 查找 NFT 交易 → 搜索 nft - 查找某个代币 → 直接输入 代币合约地址

13.2 📌 2. 通过 SQL 查询 information_schema.tables

如果你希望用 SQL 查询 Dune 数据库的所有可用表，可以运行以下 SQL：

SELECT table_schema, table_name
FROM information_schema.tables
WHERE table_schema LIKE 'erc20%'  -- 只搜索 ERC-20 相关表
ORDER BY table_schema, table_name;

📌 说明： - information_schema.tables 存储了所有 Dune 数据库的表结构。 - table_schema 是数据库名，如 uniswap_v3_ethereum、erc20_ethereum。 - table_name 是具体的数据表名，如 swaps、evt_transfer。

🔹 如果想查找所有 NFT 相关表，可以修改 SQL 为：

WHERE table_schema LIKE 'nft%'

🔹 如果想查找所有 DeFi 相关表，可以修改 SQL 为：

WHERE table_schema LIKE 'defi%'

13.3 📌 3. 查看 Spellbook 文档

如果你查找的是 Dune Spellbook 相关的魔法表，可以： 1. 访问 Spellbook GitHub 仓库（https://github.com/duneanalytics/spellbook） 2. 打开 docs/ 目录，查找对应的数据表说明。

或者在 Dune 上搜索：

SELECT *
FROM spellbook_metadata.tables
WHERE table_schema LIKE '%spellbook%'

📌 这个查询会列出所有 Spellbook 相关的魔法表。

13.4 📌 4. 通过 SHOW TABLES 快速查找

如果你想查找某个特定数据库（Schema）下的表，可以使用：

SHOW TABLES IN uniswap_v3_ethereum;

📌 示例： - 想查找 Uniswap V3 交易数据 → SHOW TABLES IN uniswap_v3_ethereum - 想查找 Aave 借贷数据 → SHOW TABLES IN aave_v2_ethereum - 想查找 NFT 交易数据 → SHOW TABLES IN nft_ethereum

13.5 📌 5. 在 Dune 社区或 GitHub 讨论区搜索

如果你不确定表的名字，可以： 1. 访问 Dune 的 Discord 频道，在 #spellbook 询问。 2. 在 GitHub 的 Spellbook 讨论区（https://github.com/duneanalytics/spellbook/discussions）搜索相关的主题。 3. 官方文档 : https://docs.dune.com/data-catalog/curated/evm/asset-tracking/balances

13.6 📌 6. 结合 Dune Dashboards

很多 Dune Dashboards（仪表盘） 公开了常用的魔法表，你可以： 1. 访问 Dune Dashboard 目录（https://dune.com/browse/dashboards） 2. 搜索关键词（如 “Uniswap”, “Aave”, “DeFi”, “NFT”） 3. 查看仪表盘 SQL 代码，找到使用的表。

13.7 📌 7. 示例查询

如果你想快速找到 Uniswap V3 的交易数据表：

SHOW TABLES IN uniswap_v3_ethereum;

如果你想查询所有 ERC-20 代币的交易表：

SHOW TABLES IN erc20_ethereum;

如果你想查找 Spellbook 相关表：

SELECT * FROM spellbook_metadata.tables;

13.8 📌 结论

方法 适用场景 速度
Data Explorer 搜索 查找 Dune 现有数据表 ⭐⭐⭐⭐⭐
SQL 查询 information_schema.tables 列出所有可用表 ⭐⭐⭐⭐
SHOW TABLES IN 只查找特定协议数据 ⭐⭐⭐⭐⭐
查看 Spellbook 文档 查找 Dune Spellbook 相关数据 ⭐⭐⭐
Dune Dashboards 通过别人写的仪表盘找到表 ⭐⭐⭐

🚀 最快的方法：直接在 Dune “Data Explorer” 搜索 或 使用 SHOW TABLES！
